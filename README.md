# [@expo/exotic](https://snack.expo.io/@bacon/exotic)

A game engine for 3D Expo Games

### Installation

```bash
yarn add @expo/exotic
```

### Usage

Import the library into your JavaScript file:

```bash
import Exotic from '@expo/exotic';
```
